# CoPay

CoPay — дипломный проект: веб-приложение для **совместного учета расходов**

## Описание проекта

Приложение помогает вести общий бюджет: добавлять траты, распределять их между участниками, отслеживать баланс по комнатам/группам и получать понятную картину по текущим расходам.

## Технологический стек

- **Backend:** Python, Django, Django REST Framework
- **Frontend:** React, Vite, TypeScript
- **OCR-сервис:** Python EasyOCR
- **Инфраструктура:** Docker, Docker Compose
- **База данных:** PostgreSQL

## Локальное клонирование и запуск

### 1) Клонировать репозиторий

```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd coPay
```

### 2) Подготовить переменные окружения

Создайте и заполните `.env`

### 3) Запустить проект через Docker Compose

```bash
docker compose up --build
```

### 4) Открыть приложение

- Frontend: `http://localhost:5173`
- Backend API: `http://localhost:8000`
- OCR-сервис: запускается вместе с остальными контейнерами через `docker compose`

### 5) Остановить контейнеры

```bash
docker compose down
```
