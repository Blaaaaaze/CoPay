# CoPay

CoPay — дипломный проект: веб-приложение для **совместного учета расходов**

## Описание проекта

Приложение помогает вести общий бюджет: добавлять траты, распределять их между участниками, отслеживать баланс по комнатам/группам и получать понятную картину по текущим расходам.

## Технологический стек

- **Backend:** Python, Django
- **Frontend:** React, Vite, TypeScript
- **OCR-сервис:** Python EasyOCR, FastAPI
- **Инфраструктура:** Docker, Docker Compose
- **База данных:** SQLite
